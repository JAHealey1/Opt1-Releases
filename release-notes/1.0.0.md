# Opt1 1.0.0

First public release.

## What's new

- Detects RuneScape clue scrolls, lockboxes, towers, puzzle boxes, map clues,
  Celtic knots, and elite compass triangulation directly from the live game
  window.
- Solutions are drawn as a transparent overlay on top of the game.
- Background "Check for Updates…" via Sparkle delivers future patches without
  a manual download.

## System requirements

- macOS 14 (Sonoma) or newer
- Apple Silicon or Intel
- Screen Recording and Accessibility permissions (granted on first launch)

## Install

- **Homebrew**: `brew tap JAHealey1/opt1 && brew install --cask opt1`
- **Direct download**: grab `Opt1-1.0.0.dmg` from the
  [GitHub Release](https://github.com/JAHealey1/Opt1-Releases/releases/tag/v1.0.0).
