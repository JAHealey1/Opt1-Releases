# Opt1 1.0.3

## What's new

- Significantly improved Celtic Knot detection and solving.
- Celtic knot no longer requires inversion for every puzzle.
- Added an additional set of auto-triangulation points for Arc compasses.
- Fixed lockbox and towers puzzles not detecting on larger screens.
- Experimental support for RS3 in game UI scaling (tell us your scale in settings!).
- The map in the compass window now zooms in on any calculated intersection.
- Signficantly improved Scan clue detection.
- Experimental support for scan clue pathing guidance.
- Your choice to 'show all spots' in the compass window is now remembered.
- Fixed some teleport locations showing in the wrong place.
- Dig spots and teleport locations now show on the Arc map.
- General improvements to matching of clue text.
- Compass window is now moveable.
- Scan window is now resizeable.
- Aligned themeing across all windows.
- Release notes should now display properly.
