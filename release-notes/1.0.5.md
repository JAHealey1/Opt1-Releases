# Opt1 1.0.5

## What's new

- Debug mode is now on by default. You can opt out in settings.
- Opt1 is now open-source
- Improved scan guidance by adding pathing information to the scan optimiser (experimental)
- Compass beams are no longer hidden when the map is zoomed in
- Scan guidance UI (experimental) now shows teleport information if next spot is a teleport
- Added the ability to mark a teleport item as not owned


## Fixes

- Fixed erroneous matches of 'treasure' in chat log text as clue scroll title
- Fixed wilderness crater scan matching against deep wilderness scan
- Added some missing teleports to dorgesh-kaan
